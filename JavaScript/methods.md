## String
```javascript
var myStr = 'mahdi';
console.log(myStr.length);  // 5
console.log(myStr[0]);  // m
// log the last character
console.log(myStr[myStr.length - 1]);  // i
// log the third character from the end
console.log(myStr[myStr.length - 3]);  // h
```
